Registro en Pilas y Colas

Descripción

Este proyecto implementa un sistema de registro de mascotas y dueños, desarrollado en C++ y ejecutado en la consola. Además de gestionar los registros, el programa permite ordenar y manipular los datos utilizando estructuras de datos como pilas y colas. Asimismo, incluye ejemplos prácticos de árboles binarios de búsqueda (ABB) y árboles balanceados AVL para demostrar su funcionamiento.

Características Principales:

Gestión de Registros:
-Registro de Dueños y Mascotas: El sistema permite ingresar y almacenar la información básica de dueños y sus mascotas, como nombre, dirección, raza, y edad de la mascota. Estos datos se gestionan de manera eficiente y se pueden visualizar cuando sea necesario.
-Búsqueda y Modificación: Los usuarios pueden buscar registros específicos y realizar modificaciones si es necesario.

Ordenamiento y Estructuras de Datos:
- Pilas (Stacks): Los registros pueden ordenarse y visualizarse en forma de pila, utilizando la estructura LIFO (Last In, First Out), donde el último elemento ingresado es el primero en ser procesado o eliminado.
-Colas (Queues): Alternativamente, los datos se pueden manejar como una cola, empleando la estructura FIFO (First In, First Out), donde el primer elemento ingresado es el primero en ser procesado o eliminado. Esto es útil para simular escenarios como listas de espera.

Árboles Binarios:
-Árbol Binario de Búsqueda (ABB): El proyecto incluye un ejemplo práctico de un árbol binario de búsqueda, donde los registros se insertan de manera que cada nodo a la izquierda es menor que el nodo padre y cada nodo a la derecha es mayor.
- Árbol AVL (Adelson-Velsky y Landis): Para demostrar la eficiencia en la búsqueda y el balanceo de datos, el sistema también implementa un árbol AVL, que es una variante del ABB que se auto-balancea para mantener una altura mínima, garantizando operaciones de inserción, búsqueda y eliminación más eficientes.

Interfaz en Consola:
El programa se ejecuta completamente en un entorno de consola, ofreciendo una experiencia interactiva donde los usuarios pueden navegar por las diferentes opciones, como añadir registros, ordenar datos, y visualizar los árboles, a través de menús sencillos y comandos directos.

  • Función 2: Orden en Filas y Colas.
  • Función 3: Graficas de árboles ABB y AVL.
  • Función 4: Edición de datos guardados.
