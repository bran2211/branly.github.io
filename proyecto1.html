Programa Sistema Operativo

Descripción

Programa que tiene herramientas funcionales para el manejo de actividades y rendimiento para el sistema de windows. Este proyecto consiste en el desarrollo de una aplicación en C# que proporciona una interfaz gráfica para la monitorización y gestión del sistema operativo Windows. La aplicación permite a los usuarios obtener información detallada sobre la versión del sistema operativo, gestionar procesos activos, monitorizar el uso de recursos del sistema y visualizar las conexiones de red en tiempo real. Metodología: Desarrollo de la Interfaz Gráfica:
Utilizando Windows Forms, se diseñó una interfaz gráfica intuitiva que facilita la interacción del usuario con las diversas funcionalidades de la aplicación.

Acceso a la Información del Sistema:
✔Versión de Windows: Se emplearon clases del espacio de nombres System.Environment para recuperar y mostrar información sobre la versión del sistema operativo.
✔Procesos Activos: Utilizando la clase System.Diagnostics.Process, se implementó la funcionalidad para listar los procesos activos y controlar su estado (pausar y detener).
✔Monitor de Recursos: Se utilizaron las clases del espacio de nombres System.Diagnostics y otras API del sistema para recuperar estadísticas en tiempo real sobre el uso de recursos del sistema, como CPU y memoria.
✔Conexiones de Red: Se accedió a la información de las conexiones de red mediante la clase System.Net.NetworkInformation, mostrando detalles relevantes de cada conexión activa.

Implementación de Funcionalidades de Gestión:
✔Pausar y Detener Procesos: Se utilizaron métodos del objeto Process para suspender y terminar procesos, integrando controles en la interfaz para realizar estas acciones de manera segura.
✔Monitor de Recursos: Se implementaron gráficos y tablas que reflejan el uso de recursos en tiempo real, similar a la funcionalidad del Monitor de Recursos de Windows.
✔Visualización de Conexiones de Red: Se desarrolló una vista que muestra la lista de conexiones de red activas, permitiendo a los usuarios ver detalles específicos de cada conexión.

  • Función 1: Verificar versión de Windows.
  • Función 2: Verificar procesos activos. "Pausarlos o detenerlos"
  • Función 3: Verificar conexiones de redes en tiempo real
  • Función 4: Desplegar monitor de recursos de windows
