Calculadora Permutaciones

Descripción

Este proyecto consiste en el desarrollo de una aplicación en Java diseñada para calcular y demostrar los primeros siete teoremas de permutación. La aplicación permite a los usuarios ingresar diferentes conjuntos de datos y aplicar los teoremas para obtener resultados precisos, facilitando el estudio y comprensión de la combinatoria.

Características Principales:

Cálculo de Teoremas de Permutación:
Teorema 1: Permutaciones de un conjunto de n elementos, calculando n!.
Teorema 2: Permutaciones de n elementos con repetición, calculando n!/k1! * k2! * ... * kr! donde k1, k2, ..., kr son las frecuencias de los elementos repetidos.
Teorema 3: Permutaciones circulares, calculando (n-1)! para un conjunto de n elementos dispuestos en un círculo.
Teorema 4: Permutaciones de n elementos cuando k elementos específicos están fijos, calculando (n-k)!.
Teorema 5: Permutaciones de un conjunto con restricciones, donde ciertos elementos deben seguir ciertas reglas, como estar en posiciones específicas.
Teorema 6: Permutaciones con elementos indistinguibles, similar al teorema 2 pero aplicado en situaciones particulares con restricciones adicionales.
Teorema 7: Permutaciones de un conjunto que incluye elementos seleccionados de otros conjuntos, donde se calculan permutaciones con inclusión o exclusión de ciertos elementos.

Interfaz de Usuario:
- Gráfica Sencilla: El programa cuenta con una interfaz gráfica de usuario (GUI) intuitiva, diseñada para facilitar la entrada de datos y la visualización de resultados.
-Entrada de Datos: Los usuarios pueden ingresar el número de elementos y otras variables relevantes para los cálculos, como la cantidad de elementos repetidos o las restricciones específicas.
-Visualización de Resultados: Los resultados de cada teorema se muestran de manera clara y detallada, permitiendo al usuario ver el proceso de cálculo paso a paso.

Funciones Educativas:
Explicación de Teoremas: Cada teorema viene acompañado de una breve explicación teórica, ayudando a los usuarios a comprender los principios matemáticos detrás de los cálculos.
Optimización de Algoritmos: Los algoritmos de cálculo están optimizados para manejar grandes números y conjuntos de datos, garantizando resultados precisos y rápidos.
Manejo de Excepciones: El programa cuenta con manejo de excepciones para asegurar que las entradas inválidas o los cálculos que exceden los límites matemáticos sean manejados de manera adecuada.

  • Función 1: Analizar cada teorema.
  • Función 2: Resolver los problemas de los teoremas.
